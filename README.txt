INTRODUCTION:
------------
This is a module which provides a "Cancel" button in the taxonomy add/edit form. Cancel button can be enabled for specific vocabulary.

INSTALLATION:
-------------
1. Copy 'taxonomy_term_cancel_button' folder to modules directory.
2. At admin/modules enable the 'taxonomy_term_cancel_button' module.

CONFIGURATION:
-------------
1. Once the module is activated, go to admin/config/user-interface/taxonomy-cancel
2. Select the vocabularies which you want to add cancel button on add/edit page (Make sure that you have added vocabularies before this step)
3. Enter the cancel button label
4. Select redirect 'Action on cancel' button from the drop down

UN-INSTALLATION:
----------------
To un-install the module,
1) Go to admin/modules
2) Find 'taxonomy_term_cancel_button' in USER INTERFACE package
3) Un-check the checkbox and save.
4) Then go to admin/modules/uninstall
5) Un-check the checkbox and click 'uninstall'.
